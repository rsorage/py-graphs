# Graphs Lib for Python

## Folder structure

```
.
├── Makefile
├── lib                          # Lib source folder
│   ├── __init__.py
│   └── graph_simple.py
└── test
    └── graph_simple_test.py     # Unit tests
```

## Unit tests

To run the unit tests on the command line:

```bash
make run-tests
```

It's also possible to use the Visual Studio Code extension for Python Testing.
